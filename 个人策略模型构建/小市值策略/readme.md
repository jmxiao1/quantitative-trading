# 2026年5月4日
### 更新version0_1
增加了roe与roa进行选股，最大回撤减小，但收益降低
1. 出现问题：
- 严重跑输基准与踏空： 策略收益 27.05%，而基准收益高达 84.80%。尤其是在图表末尾（2025年下半年），红线（基准）呈现出爆发式的直线拉升主升浪，但蓝线（策略）反应极度迟钝，几乎是一条横线，完全踏空了这波大行情。
- Alpha 值为负 (-0.02)： 这意味着你的策略非但没有创造超额收益，反而因为复杂的交易摩擦或选股偏差，跑输了市场的自然表现。
- 夏普比率极低 (0.01)： 几乎为零的夏普比率说明策略承担了高昂的风险，却几乎没有获得任何超额的无风险回报。
- 防守失效 (最大回撤 40.99%)： 结合前面的高门槛财务过滤（ROE/ROA）和交易冷却，原本是为了防守，但在 2024 年初前后的微盘股踩踏行情中，策略依然遭遇了近半的腰斩回撤，说明单纯的基本面指标挡不住系统性的流动性危机。
2. 策略失效原因：
这种“看似逻辑严密，实则业绩惨淡”的现象，通常由以下几个机制冲突导致：
- “剔除涨停”在牛市变成了“拒绝盈利”： 在图表末尾那种极端的单边牛市中，市场的赚钱效应高度集中在主升浪和龙头股上。剔除 30 天内涨停的股票，直接导致你的策略完美避开了所有强势股，只能买入那些没人关注、死气沉沉的边缘冷门股。
- 绝对阈值的刻舟求剑： ROE > 10% 和 ROA > 5% 在不同的宏观周期下意义完全不同。在经济下行期，能满足这个条件的小市值公司极少；而在放水牛市初期，领涨的往往是“困境反转”型公司（当前 ROE 极差甚至为负，但预期极好）。死板的阈值让策略失去了弹性。
3. 代码与逻辑优化方向
要挽救这个策略，我们需要把原本写死的、僵化的硬规则，变成具有一定自适应能力的动态规则。
优化一：截面相对排名替代绝对阈值 (动态基本面)
不要写死 indicator.roe > 10。应该先取股票池，然后按 ROE 从高到低排序，取前 30% 或 50% 的股票，再在其中选小市值。这样可以保证无论经济好坏，你买入的都是相对最赚钱的那批公司。
优化二：引入大盘风控 (宏观择时)
在代码中加入大盘环境判断。例如，获取沪深300指数的 60 日均线：
熊市状态（指数在均线下方）： 严格执行“剔除涨停”和“30天冷却”，甚至强制空仓，以降低回撤。
牛市状态（指数在均线上方）： 必须关闭“剔除涨停”这个条件，允许策略去追逐强势股和市场热点。
优化三：动量因子的引入
在选出基本面好的股票后，不要仅仅按照市值排序。可以叠加一个“20日涨幅”指标，在小市值+高ROE的池子里，优先买入近期已经有资金开始关注、呈现多头排列的标的，解决“买入后不涨、长期横盘”的问题。
