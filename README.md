# quantitative trading

> 记录弱鸡大学生的量化交易学习之路 —— 从数据挖掘到策略实现，从回测框架到实盘思考

## 📖 项目简介

本项目旨在记录个人系统化地整理量化交易学习过程中的知识、代码与实践项目。适合对量化交易感兴趣的初学者参考与交流，如果存在错误或者不足请指正，感谢大佬们的建议与指导。

## 🎯 学习目标

- 掌握金融数据处理与分析的基本方法
- 理解常见量化策略逻辑并实现回测
- 搭建可扩展的回测与研究框架
- 探索机器学习在量化中的应用
- 积累可复现的策略研究与工程经验
- 尝试模拟回测与含未来函数的模型项目复现
- 希望有一天能够实现自己的量化交易策略并运用到实际交易中
