1，财务因子介绍
财务因子：评价企业的基本面情况，通常包括成长类因子，规模类因子，价值类因子以及质量类因子
- 成长类因子：在财务因子选股中，常用方法是选用成长类因子进行选股。成长类因子包括营收因子和利润因子
- 规模类因子：规模类因子反映公司规模情况，主要用于体现市值大小对投资收益的影响。规模类因子包括总市值，流通市值，总股本，流通股本
- 价值类因子：价值投资是一个久经考验的投资策略，惯例是购买那种相对低价的股票，转化为在基本面标准度量股息，账面价值，利润，现金流或其他公司价值的方法。价值类因子包括市净率，市销率，以及市盈率
- 质量类因子： 质量类因子指与股票的财务质量，资本结构相关的因子。质量类因子包括净资产收益率，以及总资产收益率
---
2. 营收因子与利润因子
- 营收因子：营收因子是衡量公司成长性的一个重要指标，通常用于判断公司未来的盈利能力和发展潜力。营收因子主要包括营业收入同比增长率，营业收入环比增长率，营业总收入
- 利润因子：利润因子是衡量公司盈利能力的一个重要指标，通常用于判断公司的盈利水平和盈利能力。利润因子主要包括净利润同比增长率，净利润环比增长率，营业利润率，销售净利润，销售毛利率
---------
3. 营收因子详述
营收因子：营收因子是衡量公司成长性的一个重要指标，通常用于判断公司未来的盈利能力和发展潜力。营收因子主要包括营业收入同比增长率，营业收入环比增长率，营业总收入
- 营收因子计算公式：
    - 营业收入同比增长率 = (当期营业收入 - 上期营业收入) / 上期营业收入 * 100%     上期营业收入指上一年度/季度/月度同期营业收入，此处指上一年度的同期营业收入      代码：get_fundamentals(query(indicator.inc.revenue_year_on_year).filer(查询条件)，date=[查询日期])
    - 营业收入环比增长率 = (本期营业收入 - 上期营业收入) / 上期营业收入 * 100%     环比增长率是针对上一期的，而同比增长率是相对于上一年度同一期的    代码：get_fundamentals(query(indicator.inc.revenue_annual).filer(查询条件)，date=[查询日期])
    - 营业总收入 = 营业收入 + 其他业务收入 net_profit_to_total_revenue 代码：get_fundamentals(query(indicator.net_profit_to_total_revenue).filer(查询条件)，date=[查询日期])
