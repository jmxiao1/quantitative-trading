# 今日学习压力支撑指标，包括BOLL,MIKE,XS指标

1. BOLL指标
BOLL指标，即布林带，又称布林线指标，由约翰 布林创造，其利用统计原理，求出**股价的标准差及其信赖区间**，从而确定**股价的波动范围及未来走势**，利用**波带**显示股价的安全高低价位，因而也被称为布林带。其**上下限范围不固定**，随股价的滚动而变化
- 计算方法  中间线=N日收盘价的移动平均线
            上限=中间线+2×SD(标准差)
            下限=中间线-2×SD(标准差)
- 这里的标准差指总体标准差，取N日收盘价的均值计算
- N值一般取20
- 应用方法  当股价触及布林带的上轨时，表明股价已经超买，有回调的可能；当股价触及布林带的下轨时，表明股价已经超卖，有反弹的可能
-           股价上升穿越布林线上限时，回档概率大；股价下降穿越布林线下限时，反弹概率大
-           带宽变宽，股价波动幅度加大
-           带宽变窄，股价波动幅度减小
-           带宽变窄时，表示变盘在即

- 代码     from  jqlib.technical_analysis import *
-           Bollinger_Bands(security_list,check_date,timeperiod=20,nbdevup=2,nbdevn=2,unit='1d',include_now=True,fq_ref_date=None)
-           timeperiod:计算周期
-           nbdevup:计算上轨线时所用的指标，中轨线+X倍的标准差
-           nbdevn:计算下轨线时所用的指标，中轨线-X倍的标准差
- 返回 UP, MID, LOW三条线

2. MIKE指标
麦克指标，其英文全称为"Mike Base",是一种**股价波动幅度大小而变动**的压力支撑指标，设有**初级，中级，高级**三个不同级别的压力支撑线，分别用**蓝色，绿色，红色**表示，用图标方式直接显示压力，支撑的位置。
- 计算方法  先算出真实值TPY,TPY=(最高价+最低价+收盘价)/3  
-          三条初级，中级，高级压力线分别为WR,MR,SR
-          WR=TPY+(12日最高价-12日最低价)
-          MR=TPY+2*(12日最高价-12日最低价)
-          SR=TPY+3*(12日最高价-12日最低价)
-          三条初级，中级，高级支撑线分别为WL,ML,SL
-          WL=TPY-(12日最高价-12日最低价)
-          ML=TPY-2*(12日最高价-12日最低价)
-          SL=TPY-3*(12日最高价-12日最低价)

MIKE指标的应用方法
-           MIKE指标共有6条曲线，上方三条压力线，下方三条支撑线
-           当股价脱离盘整，朝下跌的趋势前进时，股价下方三条"下限"为其支撑参考价
-           当股价脱离盘整，朝上涨的趋势前进时，股价上方三条"上限"为其压力参考价
-           当股价往压力线方向涨升时，其下方支撑线不具备参考价值
-           股价突破初级压力，是短期走强信号。
-           股价触及强力压力，意味着可能已进入超买区，短期有回调风险

-           当股价往支撑线方向跌落时，其上方压力线不具备参考价值
-           股价跌破初级支撑，是短期走弱信号。
-           股价触及强力支撑，意味着可能已进入超卖区，短期或有反弹机会

3. XS指标
薛斯通道指标是判断股价运行区间及相应的压力，支撑的趋势性指标。属于短期指标之一。由美国著名技术分析家薛斯（Charles C. Schwaiger）所创，故称薛斯通道指标。在薛斯通道中股价实际上是被**短期小通道**包容着在**长期道**中上下运行，基本买卖策略是当**短期小通道接近长期大通道时，预示着趋势的近期反转**
指标运用法则
- 上升趋势中（长期大通道向上）
- 当股价经历一波上涨后回落至内下轨（紫线）附近并获得支撑，同时成交量再次放大时，这是一个很好的买入点，常被称为“空中加油”
- 当内上轨（黄线）从下方向上穿越外上轨（白线） 时，表明股价进入强力拉升阶段。只要股价收盘价不跌破内上轨（黄线），就可以继续持股待涨
- 如果股价快速上涨并触及或远离外上轨（白线），则进入超买区域，应警惕回调风险，考虑逐步卖出

- 下降趋势中（长期大通道向下）
- 在下跌趋势中，股价的每一次反弹到外上轨（白线） 附近，都是强阻力位，是减仓或卖出的良机
- 当股价持续下跌并跌破外下轨（绿线） 后，往往进入超卖区。此时不应立即买入，而应等待股价重新站回外下轨（绿线）上方，且长期通道有走平迹象时，再考虑是否见底

- 横盘震荡中（长期大通道走平）
- 下轨买，上轨卖：当股价触及或跌破外下轨（绿线） 时，是买入时机；当股价触及或突破外上轨（白线） 时，是卖出时机
- 突破跟进：如果股价放量上涨，带动短期通道强力上穿长期通道，且长期通道开始向上转向，这可能是震荡结束、趋势启动的信号，应转为持股
