# 今日继续学习风险指标，包括索提诺比率，信息比率，策略波动比率
---
1.索提诺比率
定义：索提诺比率（Sortino Ratio）是衡量投资组合风险的一种指标，表示**每承担一单位的下行风险将获得多少超额回报**，它通过计算投资组合的下行风险来衡量投资组合的风险。
算法：与夏普比率类似，但索提诺比率运用**下行波动率**而不是总标准差，以区别不利和有利的波动。**这一比率越高，表明承担相同单位下行风险时能获得更高的超额回报率**。索提诺比率可以看做是夏普比率在衡量股票风险的一种修正方式。
- 计算方式：Sortino Ratio = (Rp - Rf) / Downside Deviation
- Rp：投资组合的预期回报率，或策略年化收益率
- Rf：无风险利率(默认0.03)
- Downside Deviation：投资组合的下行波动率

索提诺比率回测：在聚宽回测界面单击左侧“索提诺比率”可查看量化策略的索提诺比率信息

2.信息比率
定义：信息比率（Information Ratio）是衡量**单位超额风险带来的超额收益**，它表示每承担一单位的超额风险将获得多少超额回报。信息比率可以看做是索提诺比率在衡量股票风险的一种修正方式。
算法：**信息比率越大，说明该策略单位跟踪误差所获得的超额收益越高**，因此，信息比率较大的策略的表现要优于信息比率较低的基准。合理的投资目标**在承担适度风险下，尽可能追求高信息比率**。
- 计算方式：Information Ratio = (Rp - Rf) / Tracking Error
- Rp：投资组合的预期回报率，或策略年化收益率
- Rf：无风险利率(默认0.03)
- Tracking Error：策略与基准每日收益差值的年化标准差，即策略与基准的协方差除以基准的标准差

信息比率回测：在聚宽回测界面单击左侧“信息比率”可查看量化策略的信息比率信息

3.策略波动比率
定义：策略波动比率（Volatility Ratio）是衡量股票或者衍生品价格波动范围的指标，也通常被用来**衡量风险**，表示**每承担一单位的波动风险将获得多少超额回报**。一般认为，**高波动率代表高风险，高风险对应高收益**的可能。策略波动比率可以看做是索提诺比率在衡量股票风险的一种修正方式。
- 计算方式：Volatility Ratio = (Rp - Rf) / Volatility or sqrt[(250/n-1)*sum((Rp-Rf)^2)]
- Rp：投资组合的预期回报率，或策略年化收益率
- Rf：无风险利率(默认0.03)
- Volatility：策略每日收益的年化标准差
- n：策略执行天数

策略波动比率回测：在聚宽回测界面单击左侧“策略波动比率”可查看量化策略的策略波动比率信息
