# 今日学习自定义因子，包括自定义因子介绍，因子计算，单因子分析，以及代码实战
---
1. 自定义因子介绍
自定义因子类包括:三个基本属性，分别为name,max_window和dependencies;以及一个核心函数，cal()
流程：**构造因子属性**->**实现因子计算**->**因子分析**
- 属性介绍
- name：因子名称，因子命名只能由字母，数字和下划线组成，并且第一个字符不能是数字，另外不能与python中的关键字相同，也不能和基础因子冲突
- max_window：因子计算过程中需要用到的数据窗口大小，即因子计算过程中需要用到多少天的数据的最长时间窗口，返回日线数据
- dependencies：自定义因子依赖的基础因子名称，如high,low,price

2. 因子计算
实现方法：实现Factor类内置cal()函数 calc(self,data)
- data:字典对象，key属性是dependencies中的因子名称，value属性是因子对应的数据表(pandas.DataFrame格式)
- 返回：Pandas.Series:index属性是股票代码，value属性是因子值

3.单因子分析
实现方法:调用analyze_factor函数
- analyze_factor(factor,start_date,end_date,industry,universe,quantiles,periods,weight_method,use_real_price,skip_paused,max_loss,factor_dep_definitions)
- factor:因子对象.可输入三种类型的值：
- 1.pandas.DataFrame格式的因子值，columns为股票代码，index为日期的DatetimeIndex或str,
- 2.pandas.Series格式的因子值，index为日期和股票代码的MultiIndex,
- 3.Factor的子类，自定义因子

- start_date,end_date:开始和结束日期
- industry:行业分类，默认为'jq_l1'
- universe:股票池，可输入两种值：为str时认为输入的是一个指数，股票池为这个指数的成分股，为list时认为输入的是一个股票池
- quantiles:分位数，默认为10
- periods:计算周期，默认为[1,5,10,20,60,120,250]
- max_loss:因重复值或nan值太多而无效的因子值的最大占比，默认为0.25
- weight_method:权重计算方法，默认为等权；avg:按平均加权，mktcap：按市值加权
- use_real_price:是否动态复权，默认为False
- skip_paused：是否跳过停牌，默认为False
- factor_dep_definitions:主因子的依赖因子的列表，默认为空列表
