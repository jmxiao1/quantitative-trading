# 今日学习因子分析结果，包括结果展示函数，结果分析解读与代码实战
---
1. 结果展示函数
实现方式： create_full_tear_sheet(demeaned=False,group_adjust=False,by_group=False,turnover_periods=None,avgretplot=(5,15),std_bar=False)
参数说明：
- demeaned：是否使用超额收益计算
- group_adjust：是否使用行业中性化收益计算
- by_group：是否按行业展示
- turnover_periods：调仓周期
- avgretplot：tuple因子预测的天数(计算过去的天数，计算未来的天数)
- std_bar：是否展示因子值标准差

2. 结果分析解读
收益分析:因子收益分析。包括7种常见的收益分析
IC分析：指information coefficient，即因子IC值，代表预测值和现实值之间的相关性，通常用以评价预测能力。取值在-1到1之间，越接近1越好，表示预测能力越强。
换手率分析：因子的换手率是在不同的时间周期下，观察因子个分位中的个股的进出情况

收益分析：在收益分析中，分位数的平均收益，各分位数的累积收益，以及分位数的多空组合收益三方面观察因子的表现。**第一分位数的因子值最小，第五分位数的因子值最大**，因此第一分位数的多空组合收益是空仓收益，第五分位数的多空组合收益是多头收益。
- 分位数收益:表示持仓1，5，10天后，各分位数可以获得的平均收益
- 分位数累积收益：表示持仓1，5，10天后，各分位数可以获得的累计收益
- 分位数多空组合收益：做多五分位(因子值最大)，做空一分位(因子值最小)的投资组合收益

IC分析:一般有两种方法，即**normal IC**和**rank IC**。normal IC是因子值和实际收益之间的相关系数，rank IC是因子值的排序和实际收益的排序之间的相关系数。**rank IC的取值范围是-1到1，越接近1越好**，表示预测能力越强。聚宽平台默认使用rank IC。
- normal IC：因子载荷与因子收益之间的相关系数
- rank IC：因子载荷的排序与因子收益的排序之间的相关系数
考虑到单日IC的波动较大，聚宽提供IC的月度移动平均线作为参考

换手率分析:
- 计算方法举例:某因子第一分位持有的股票数量为30支，一天后有一支发生变动，其换手率为1/30 *100%=3.33%
对于5日，10日的换手率，在每日都会对比当日1，5分位的成分股与5，10日前该分位数的成分股的变化进行计算
换手率价值分析:
- **因子稳定性的体现**:换手率低的因子，因子值在时间序列成面的持续性更好
- **衡量交易成本**：在实际交易中，假设我们要维护投资组合的一种暴露恒定，对于高换手率的因子，则需要进行更多的交易。交易中的税费和滑点，也会吞噬我们的部分利润
