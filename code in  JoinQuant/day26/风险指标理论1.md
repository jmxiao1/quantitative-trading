# 今日学习量化回测所用到的风险指标，包括风险指标介绍，Alpha,Beta,以及夏普比率
---
1. 风险指标介绍
定义:即股票投资收益率的**不确定性**通常称之为**风险**，具体指股票市场的一些未知的，不可预测的因素对股票投资收益造成不确定的影响，可能是正面影响收益率，也可能是反面的背离。风险指标是**衡量股票投资收益不确定性的指标**，是对**风险的量化评价**。风险指标越高，说明股票投资收益的不确定性越大，风险越大。
特点：风险指标有利于投资者对策略进行一个客观的评价。无论是回测还是模拟，所有风险指标都只会根据**每天收盘后的收益**计算并每日更新，**不会根据当天交易情况与盘中的收益情况实时更新**，因此风险指标只反映**策略的历史表现，不能反映策略的未来表现**。

2. Alpha
定义：Alpha是**投资者获得与市场波动无关的回报**。阿尔法系数，是基金/投资的绝对回报和按照Beta系数计算的预期回报之间的差额。阿尔法收益与风险相关性很低，因此它被投资者视为衡量投资业绩的“最佳”指标。阿尔法系数越高，说明基金/投资相对于市场表现得越好。阿尔法系数越低，说明基金/投资相对于市场表现越差。例如，一个策略的阿尔法为1，则说明该策略相对于市场表现好1倍，如果阿尔法为-1，则说明该策略相对于市场表现差1倍。
算法：**绝对收益**或**超额收益**是基金/投资的**实际收益减去无风险投资收益**。阿尔法系数是**超额收益**与**基金/投资的风险**之间的比率。阿尔法系数越高，说明基金/投资相对于市场表现得越好。阿尔法系数越低，说明基金/投资相对于市场表现越差。
- 计算公式：Alpha =Rp-[Rf+Beta(p)*(Rm-Rf)]
- Rp:策略收益
- Rf:无风险收益
- Rm:市场收益
- Beta(p):策略的贝塔值
- Alpha=投资年化收益率-[无风险回报率+Beta*(基准市场年化收益率-无风险回报率)]
Alpha回测：在聚宽回测界面单击左侧“阿尔法”可查看量化策略的Alpha信息

3. Beta
定义：Beta是表示**投资的系统性风险**，反映了**策略对大盘变化的敏感性**。Beta值越大，策略对大盘变化的敏感性越强，Beta值越小，策略对大盘变化的敏感性越弱。Beta值越大，策略的波动性越大，Beta值越小，策略的波动性越小。例如，一个策略的Beta为1.5，则大盘涨1%时，策略的收益将涨1.5%，如果大盘跌1%，策略的收益将跌1.5%。
算法：Beta系数是**基金/投资相对于市场波动的敏感性**。Beta系数越大，说明基金/投资相对于市场波动的敏感性越强，Beta系数越小，说明基金/投资相对于市场波动的敏感性越弱。Beta系数越大，说明基金/投资的风险越大，Beta系数越小，说明基金/投资的风险越小。
- 计算公式：Beta = Cov(Dp,Dm)/Var(Dm)
- Dp:策略每日收益
- Dm:市场基准每日收益
- Cov(Dp,Dm):策略收益和市场收益的协方差
- Var(Dm):市场每日收益的方差
Beta回测：在聚宽回测界面单击左侧“贝塔”可查看量化策略的Beta信息

4. 夏普比率
定义：夏普(sharp)表示**每承受一单位总风险，会产生多少超额报酬**，可以同时对策略的收益与风险进行综合考虑
算法：夏普指数代表投资人每多承担一分风险，可以拿到较无风险报酬率(定存利率)高出多少的报酬；若为正数，代表**投资承担报酬率波动风险有正的回馈**；若为负数，代表**承担风险但报酬率反而不如银行利率，无参考意义**。夏普比率越高，说明策略的收益与风险成正比，夏普比率越低，说明策略的收益与风险不成正比。
- 计算方法 ：夏普比率=（Rp-Rf）/σp
- Rp:策略收益
- Rf:无风险收益
- σp:策略收益的标准差
- 夏普比率=（策略年化收益率-无风险回报率）/策略年化收益率的波动率
夏普比率回测：在聚宽回测界面单击左侧“夏普比率”可查看量化策略的夏普比率信息
