# 今日学习MA-RSI策略，包括量化策略介绍，策略实现思路以及代码实战
---
1. 量化策略介绍
MA-RSI策略是一种常见的量化策略，它结合了移动平均线和相对强弱指数（RSI）指标。移动平均线可以帮助我们识别趋势，而RSI指标可以帮助我们识别超买和超卖信号,从而选出上升趋势的超卖个股，同时缩小了选股范围，做简单的交易策略，达到稳定的收益
- 选股范围:选取市值大于500亿的股，排除st，科创板，创业板，退市，次新股股票
- 买卖信号： K线在MA-200均线上方，RSI-10小于25，**买入股票**，
-           RSI-10大于40，持有N天；个股下跌超过5%，**卖出股票**
