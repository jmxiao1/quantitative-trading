# 今日学习量价指标，包括OBV，VOL指标
---
1. OBV指标
OBV即能量潮指标，是葛兰维与上世纪60年代提出的。OBV线系依据**成交量的变化**统计绘制而成。OBV线为股市**短期波动**的重要判断方法，但运用OBV线应配合股价趋势予以研判分析
- 计算方法：今日OBV=昨日OBV+今日成交量 or OBV=昨日OBV+sgn+今日成交量
-           其中，sgn为**符号函数**，其值有以下公式决定：
-           sgn=1，今日收盘价>昨日收盘价
-           sgn=0，今日收盘价=昨日收盘价
-           sgn=-1，今日收盘价<昨日收盘价
- MAOBV指标：MAOBV=OBV的移动平均线，一般取5日、10日、20日、30日等
- 注意：这里的成交量是成交股票的手数，不是**成交金额**
- OBV线上升表示市场人气旺盛，股价上涨；OBV线下降表示市场人气低迷，股价下跌
- 股价一顶比一顶高，而OBV线却一顶比一顶低，说明股价上涨动力不足，暗示头部即将形成，股价将面临压力
- 股价一底比一底低，而OBV线却一底比一底高，说明股价下跌动力不足，暗示底部即将形成，股价将面临支撑
- OBV突破其N字形波动的高点次数达5次，为**短线卖点**
- OBV突破其N字形波动的低点次数达5次，为**短线买点**
**OBV不能单独使用**，必须结合股价趋势、KDJ、MACD等指标一起使用
- 代码          from  jqlib.technical_analysis import *
                OBV(security_list,check_date,timeperiod=30,unit='1d',include_now=True,fq_ref_date=None)
- 参数
-                security_list：股票代码列表
-               check_date：检查日期
-               timeperiod：计算周期，默认30
-               unit：时间单位，默认为1d
-               include_now：是否包含当前时间，默认为True
-               fq_ref_date：复权基准日期，默认为None
- 返回值：OBV值

2. VOL指标
VOL指标，即成交量指标。成交量是指个股或大盘的成交总手，在形态上用一根立式的柱子来表示。左边的坐标值与柱子的横向对应处，就是当然当时的成交总数。如当日收盘价高于或等于当天的开盘价，成交柱为红色；反之，成交柱为绿色
- 计算方法：VOL=今日成交量 or VOL=今日成交量-昨日成交量
- MAVOL指标：MAVOL=VOL的移动平均线，一般取5日、10日、20日、30日等
- 注意：这里的成交量是成交股票的手数，不是**成交金额**
- 当股价处在盘整的行情的时候，成交量突然增加的话，股价短时间内突破的可能性较高
- 当成交量出现了连续三条或者更多的成交量的时候，说明当前市场买卖频繁，股价下跌的可能性比较少
- 股价在上涨了一定时间后，成交量依旧大幅上涨，但是最后收尾的时候，趋势阴线或者比较少的阳线收尾的时候，说明当前上涨的支撑力量已经衰退，股价走势在短时间内会发生反转
- 代码          from  jqlib.technical_analysis import *
                VOL(security_list,check_date,M1=5,M2=10,unit='1d',include_now=True,fq_ref_date=None)
- 参数
-               security_list：股票代码列表
-               check_date：检查日期
-               M1：短期移动平均线周期，默认为5;M2：长期移动平均线周期，默认为10
- 返回 VOL,MAVOL1,MAVOL2
