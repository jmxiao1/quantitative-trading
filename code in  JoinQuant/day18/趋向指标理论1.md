# 今日学习趋向指标，包括其基本概念，MACD，EMV与UOS指标
---------
# 趋向指标
趋向指标(DMI)又称动向指标，其基本原理是通过分析股价在上升及下跌过程中的**均衡**(即买卖双方力量均衡点的变化情况)，即**供需关系受价格**的变动由**均衡到失衡**的循环过程，从而判断股价的走势。DMI指标主要包括ADX、ADXR、DIF、DEA、MACD、EMV、UOS等指标。
- 原理：在大多数指标中都是以每一天的收盘价走势及升幅，跌幅的累计数来计算不同的分析数据。】，其不足之处在于**忽略了每一日的高价和低价的波动幅度**。而DMI指标则是在分析股价走势时，不仅考虑了收盘价，还考虑了**每日的高低波动的幅度因数**，从而更准确地反映了股价的波动情况，分析预测未来的走势。

# MACD指标
MACD指标即平滑异同移动平均线，是由美国投资家杰拉尔德·阿佩尔斯都德(Gerald Appel)所创，属于**趋势型指标**，主要表示经过平滑处理后均线的差异程度。一般用来研判股票价格变化的方向，强度和趋势
原理：MACD的原理是通过计算快速移动平均线与慢速移动平均线之间的差离值，来反映多空双方力量的强弱对比。当快速移动平均线从下向上突破慢速移动平均线时，表明股价将上涨；当快速移动平均线从上向下突破慢速移动平均线时，表明股价将下跌。
- 计算公式：MACD=EMA(12)-EMA(26)
- 计算公式：DIF=EMA(12)-EMA(26)
- 计算公式：DEA=EMA(9)
- 计算公式：MACD柱=（DIF-DEA）*2
- 当快线DIFF由下向上突破慢线DEA时，称为**金叉**，表明买入信号
- 当快线DIFF由上向下突破慢线DEA时，称为**死叉**，表明卖出信号
聚宽平台实现方式：from  jqlib.technical_analysis import *
          MACD(security_list,check_date,SHORT,LONG,MID)
主要参数包括：
- security_list：股票代码列表
- check_date：要查询数据的日期
- SHORT：短期EMA周期
- LONG：长期EMA周期
- MID：DIF与DEA的平滑周期
- EMA 指数移动平均线，默认参数 SHORT=12，LONG=26，MID=9，公式为EMA(t)= 今日价*a+昨日EMA*(1-a),a=2/(N+1),N为周期数
# EMV指标
EMV指标即简易波动指标，是由Richard W. Arms Jr.所创，属于**趋势型指标**，主要用来衡量股价的波动幅度，从而判断股价的强弱。
原理：股票中间价的相对波动幅度是以相对成交除以相对振幅作为衡量股票中间价波动百分比的基数。EMV值增加代表成交量增加，这是价格上升的信号；EMV值减少代表成交量减少，这是价格下跌的信号。

- 计算公式：A:(今日最高-进入最低)/2
- 计算公式：B:(前日最高-前日最低)/2
- 计算公式：C:今日最高-今日最低
- 计算公式；EM=(A-B)*C/今日成交量
- 计算公式：EMV=N日内EM的累计值
- 计算公式：MAEMV=EMV的M日简单移动平均线
- 默认参数：N=14，M=9
- EMV上穿0轴，表明股价将上涨
- EMV下穿0轴，表明股价将下跌
- EMV上穿MAEMV，表明股价将上涨
- EMV下穿MAEMV，表明股价将下跌
聚宽平台实现方式：from  jqlib.technical_analysis import *
                EMV(security_list,check_date,N,M，unit='1d',include_now=True,fq_ref_date=None)
主要参数包括：
- security_list：股票代码列表
- check_date：要查询数据的日期
- N：EMV的累计周期
- M：EMV的移动平均周期
- 常见参数 N=14，M=9
- unit：时间单位，默认为‘1d’
- include_now：是否包含当前日期，默认为True
- fq_ref_date：复权基准日期，默认为None

# UOS指标
UOS指标即终极波动指标，是由Welles Wilder Jr.所创，属于**趋势型指标**，除了趋势确认及超买超卖方面的作用外，它的“突破”讯号不仅可以提供最适当的交易时机之外，更可以进一步加强指标的可靠性。
原理：UOS指标是结合了RSI、TRIX、EMA三种指标，通过计算股价的波动幅度，来反映股价的强弱。
- 计算公式：TH=今日最高价和昨日收盘价的较大值
- 计算公式：TL=今日最低价和昨日收盘价的较小值
- 计算公式：ACC1=(收盘价-TL)的N1日积累和/(TH-TL)的N1日积累和
- 计算公式：ACC2=(收盘价-TL)的N2日积累和/ (TH-TL)的N2日积累和
- 计算公式：ACC3=(收盘价-TL)的N3日积累和/ (TH-TL)的N3日积累和
- 计算公式：UOS=(ACC1*N2*N3+ACC2*N1*N3+ACC3*N1*N2)/(N1*N2+N1*N3+N2*N3)
- 计算公式：MAUOS=UOS的M日简单移动平均线
- 默认参数：N1=7，N2=14，N3=28，M=9
聚宽平台实现方式：from  jqlib.technical_analysis import *
                UOS(security_list,check_date,N1=7,N2=14,N3=28,M=6,unit='1d',include_now=True,fq_ref_date=None)
主要参数包括：
- security_list：股票代码列表
- check_date：要查询数据的日期
- N1：UOS的累计周期1
- N2：UOS的累计周期2
- N3：UOS的累计周期3
- M：UOS的统计天数
- 常见参数 N1=7，N2=14，N3=28，M=9
- unit：时间单位，默认为‘1d’
- include_now：是否包含当前日期，默认为True
- fq_ref_date：复权基准日期，默认为None
UOS短线抄底：UOS上穿50
UOS短线卖出：UOS下穿65
UOS中长线抄底：UOS上穿35
UOS中长线卖出：UOS下穿70
UOS>70,为超买；UOS<30，为超卖
 

# 总结
趋向指标主要包括MACD，EMV，UOS等指标
MACD看**大趋势，快慢线背离**最稳，但**滞后**
UOS看**多周期动量共振**，比MACD快，**抓拐点更强**，但**容易产生假信号**
EMV看**量价配合**，判断上涨下跌**轻不轻**，专看**假突破**，但**容易产生假信号**
- MACD指标用于判断股价的涨跌趋势，当快线DIFF由下向上突破慢线DEA时，称为金叉，表明买入信号；当快线DIFF由上向下突破慢线DEA时，称为死叉，表明卖出信号
- EMV指标用于衡量股价的波动幅度，EMV上穿0轴，表明股价将上涨；EMV下穿0轴，表明股价将下跌
- UOS指标用于判断股价的强弱，UOS上穿50，为超买；UOS下穿30，为超卖
