# GDX指标
GDX指标即鬼道线，是衡量黄金期货市场情绪的指标，反应的是中心线于收盘价的乖离，对趋势线做了平滑和修正处理，更精准的反应股价的运行规律
- 计算方法： AA赋值(2*收盘价+最高价+最低价)/4-收盘价的N日简单移动平均的绝对值/收盘价的N日简单移动平均 
- 输出济安线(JAX):以AA为权重收盘价的动态移动平均,指标中线
- 输出压力线:(1+M/100)*JAX，上轨
- 输出支撑线:(1-M/100)*JAX，下轨
- N为均线周期，M为通道宽度
- 判断方法：当JAX在压力线和支撑线之间运行时，表示市场情绪稳定，当JAX上穿压力线时，表示市场情绪看涨，当JAX下穿支撑线时，表示市场情绪看跌
- 使用方法：当股价上升到压力线时，卖出，当股价下穿支撑线时，买进

# JS指标
JS指标即加速线，是衡量股票市场情绪的指标，通过当前股价与一段时间以前的股价相对比来测量股价涨跌的速度来确定其运行的快慢的指标
- JS指标包含四条线：JS线以及三条JS不同周期的简单移动平均线，一般情况下，分别取5日，10日，20日的简单移动平均线，具体计算如下
- JS线：JS线=100*(收盘价-前N日收盘价)/(前N日收盘价*N)
- MAJS1=JS线的5日简单移动平均
- MAJS2=JS线的10日简单移动平均
- MAJS3=JS线的20日简单移动平均
- 这里的N日一般为5日
- 判断方法：当JS线向下交叉JSMA线，卖出；当JS线向上交叉JSMA线，买进
代码：from  jqlib.technical_analysis import *
               JS(security_list,check_date,N=5,M1=5,M2=10,M3=20,unit='1d',include_now=True,fq_ref_date=None)
- security_list:股票代码列表
- check_date:检查日期
- N:JS线计算周期
- M1:MAJS1计算周期
- M2:MAJS2计算周期
- M3:MAJS3计算周期
- unit:时间单位，默认为1d
- include_now:是否包含当前日期，默认为True
