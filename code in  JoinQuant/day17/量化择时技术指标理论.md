# 今日学习量化择时，包括其基本概念，常用方法，以及技术指标理论与分类
---------
# 量化择时
量化择时是指利用数学模型和数据分析方法(即数量化)，通过对各种宏观微观指标的量化分析，试图找到影响大盘走势的关键信息，并且对未来走势进行预测，从而指导投资者进行投资决策的过程。通俗来说，就是采用量化的方式判断买点和卖点。
量化择时通常包括以下几个步骤：
1. 数据收集：收集市场数据，包括价格、成交量、技术指标等。
2. 数据处理：对收集到的数据进行预处理，包括数据清洗、数据转换等。
3. 模型构建：根据历史数据，构建预测模型，如时间序列模型、机器学习模型等。
4. 模型评估：对构建的模型进行评估，包括模型准确率、稳定性等。
5. 模型应用：将模型应用于实际市场，进行投资决策。

# 常用方法
常用方法有趋势量化择时，市场情绪量化择时，基本面量化择时，机器学习量化择时等。
1. 时间序列分析：通过分析历史价格、成交量等数据，预测未来走势。
2. 趋势量化择时：基本思想来自于技术分析，技术分析认为趋势存在延续性，因此只要找到趋势方向，跟随操作即可。趋势择时的主要直白哦有MA,MACD,DMA等。(技术面)
3. 机器学习：利用机器学习算法，如神经网络、支持向量机等，对市场数据进行建模，预测未来走势。
4. 市场情绪分析：利用投资者的热情程度来判断大势走向，当情绪热烈时，大盘可能继续涨；当情绪冷淡时，大盘可能继续跌。常用方法有情绪指数、情绪指标，调查问卷，报告评级，舆情数据，融资融卷数据等。(消息面)

# 技术指标理论
-前提：技术指标理论基础基于三项市场假设：市场行为涵盖一切信息，价格随趋势移动，历史会重演。
技术指标是技术分析中使用最多的一种方法，通过考虑市场行为的多个方面建立一个数学模型，并给出完整的数学计算公式，从而得到一个体现证券市场的某个方面内在实质的数字，即所谓的技术指标值。
技术指标分类：
1. 基本面指标：如市盈率、市净率、每股收益等。
2. 趋向指标：技术趋向指标是识别和追踪有趋势的图形类指标，其特点是不试图捕顶和测底，如移动平均线、MACD、RSI等。 
3. 反趋向指标： 反趋向指标又称为振荡指标，是识别和追踪趋势运行的转折点的图形类指标，其特点是具有强烈的捕顶和捉底的意图，对市场转折点较敏感，如KDJ,RSI,CCI等。
4. 压力支撑指标：又称通道指标，是通过顶部轨道线和底部轨道线，试图捕捉行情的顶部和底部的图形类指标，其特点是具有明显的压力线，也有明显的支撑线，如布林线，布林通道，XS薛斯通道指标等。
5. 量价指标：量价指标是通过成交量变动分析捕捉价格未来走势的图形类指标，其特点是分析成交量与价格涨跌的关系，如OBV,VR,OBV,CCI等。
