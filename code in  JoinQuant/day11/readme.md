1. 量化选股概况
量化选股：利用数量化的方法选择股票组合，期望该股票组合能够获得超越基准收益率的投资行为，包括：
- 基本面选股：通过对一家上市公司在发展过程中面临的外部因素和自身因素进行分析，对其未来的发展前景进行预测，判断该上市公司的股票是否值得买进
- 技术面选股：利用各种技术理论或者技术分析指标来分析和预测股票的未来价格趋势
- 因子选股：通过构建各种因子模型，如价值因子、动量因子、质量因子等，来选择具有较高投资价值的股票；
- 模型选股：通过构建各种量化模型，如机器学习模型、深度学习模型等，来选择具有较高投资价值的股票。

2. 量化选股注意事项
- 分配多股，减少单股重仓的情况
- 全面研究个1股基本面，增强个股判断逻辑和支撑
- 主动投资而非被动投资
- 风险控制：量化选股需要充分考虑风险控制，确保投资组合的稳定性和安全性；
- 持续优化：量化选股需要不断优化模型和策略，以适应市场变化和投资需求。
- 只是投资手段，不是投资目的，做好亏钱准备
- 数据质量：数据质量是量化选股的基础，需要确保数据的准确性和完整性；
- 模型选择：选择合适的模型是量化选股的关键，需要根据具体情况进行模型选择和优化；
