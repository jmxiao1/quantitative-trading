# 今日学习因子分析，包括因子分析概述，因子类别与流程与常见基础因子
---
1. 因子分析概述
定义：因子分析是股票投资的常见思路。指利用“因子”或特定：特征“使各种股票收益最大化的投资方法。这些因素包括**波动性，动量，股票规模**等
原理：因子分析将多个实测变量转换为少数几个综合指标(或称潜变量),它反映的是一种**降维**的思维。通过降维将相关性高的变量聚在一起，从而减少需要分析的变量的数量，而减少问题分析的复杂性。用来确定维度数量，对标体系的维度由主观来做判断

2. 因子类别及流程
量化选股因子：多为**财务指标**，如营业利润率，销售利润率，营收环比增长率等
量化择时因子： 多为**技术指标**，如均线，换手率，波动率等

流程：**因子构造**->**因子筛选**->**因子加权**->**因子选股**->**构建股票池**->**组合优化**->**组合回测**

3. 常见基础因子
- 1.价量因子：价量因子，是指利用**get_price()**函数可以获得的价量信息，如open(开盘价)，close(收盘价)，high(最高价)，low(最低价)，volume(成交量)等
- 2.财务数据因子：财务数据因子，是指**当日可以看到的最新单季财务指标**，如pe_ratio(市盈率)，pb_ratio(市净率)，roe(净资产收益率),turnover_ratio(换手率),market_cap(股票的总市值)，circulating_market_cap(股票的流通市值)等
- 3.行业因子：行业因子，是指**当日可以看到的最新行业分类**，包括**证监会行业分类，聚宽一，二行业分类以及申万一，二，三级行业分类**等，如A01(农业),A02(林业)，B02(采矿业)，C03(制造业)等.返回值是一个哑变量，若某股票属于某行业，则返回1，否则返回0
- 4.概念因子：概念因子，即**概念板块代码**，如GN028(智能电网)，GN030(物联网)，GN092(高端装备制造)等。返回值是一个哑变量，若某股票属于某个概念，则返回1，否则返回0
- 5.指数因子：指数因子，即**指数代码**，如000001.XSHG(上证指数)，000300.XSHG(A股指数)等。返回值是一个哑变量，若某股票属于某个指数的成分股，则返回1，否则返回0
- 6.资金流因子：资金流因子，即**get_money_flow()**函数查询的数据。可以使用的字段包括：change_pct(涨跌幅)，net_amount_main(主力净额)，net_amount_xl(超大单净额)，net_pct_xl(超大单净占比),net_amount_l(大单净额)，net_pct_l(大单净占比),net_amount_m(中单净额)，net_pct_m(中单净占比),net_amount_s(小单净额)，net_pct_s(小单净占比)等
