# 今日学习量化策略回测流程，包括整体流程与步骤分解，最后是简易策略回测复现
---
1. 策略回测整体流程
包括**选择股票池和实现回测函数**->**设定回测时间段，初始金额及调仓频率**->**(在平台)历史数据载入(handle_data)和策略实现(Backtest)**->**(平台)处理订单**->**(平台)取消未完成订单**->**(平台)输出日志**->**生成回测报告**

2. 策略回测步骤分解
- 选择股票池和实现回测函数，实现handle_date函数，编写量化策略
- 设定回测开始日期和结束日期，选择初始金额及调仓频率
- 取得股票数据，调用handle_data函数，实现策略
- 下单后，根据下单时间的实际交易情况，处理订单  
- 下单后，可调用get_open_orders取得所有未完成的订单，调用cancel_order取消未完成的订单
- 回测结束，生成回测报告，画出收益曲线和基准收益曲线，列出每日持仓，交易数据指标等
