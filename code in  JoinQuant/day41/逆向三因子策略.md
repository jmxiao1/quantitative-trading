# 今日学习逆向三因子策略，包括三因子量化交易策略介绍，逆向三因子量化交易策略介绍，策略实现思路
---
1. 三因子量化交易策略介绍
三因子策略:即Fama-French三因子模型,Fama和French 1992年对美国股票市场决定不同股票回报率差异的因素的研究发现,**股票的市场的beta值不能解释不同股票回报率的差异,而上市公司的市值、账面市值比、市盈率可以解释股票回报率的差异**。Fama and French认为,上述超额收益是对CAPM中β未能反映的风险因素的补偿。
- 三因子模型的作用,在于发现了**股票的期望收益不仅仅与市场的系统风险有关,还和市值风险和账面市值比风险有关**。
- 市场风险因子：市场回报(沪深300的日度收益率)减去无风险收益率(国债的日度收益率)
- 市值因子：公司的总市值的自然对数
- 账面市值比因子：最新一季财报的账面价值与当前市值的比值(pb_ratio的倒数)

2. 逆向三因子量化交易策略介绍
传统三因子策略价值：**小市值的公司的股票平均收益率更高，市净率(M/E)率低的公司的平均收益更高**
传统三因子策略不足：并没有找到两个因子很好的解释市值溢价和价值溢价，在A股市场2016年之后的表现不好
逆三因子策略：采用反向指标替换传统三因子模型。用**VROC12、Residual，Volatility和Size**三个因子构建Zscore,每月取Zscore最低的50支股票建立多头头寸。

特质因子：Residual Volatility,残差波动率因子,又称特质因子。**Barra模型中衡量市场中不能解释的信息数据的波动情况**。在A股中效果显著
市值因子：使用Size因子,即市值规模的因子。根据研报及经验分析,**A股市场市值因子效应是存在的,小市值股票收益率更高。**
情绪因子：VROC12,即情绪因子,通过近期交易量和N日前平均交易量的差异计算

选股策略：包含所有A股股票,过滤ST股票
策略信号：每个月第一个交易日前,挑选出市值、特质因子、情绪因子最低(三个指标等权重平均计算Z-Score)的50支股票
交易方式：等权持仓
