1. 基准波动率
定义：Benchmark Volatility，简称BV，是指与某个基准资产相比，某个资产在一段时间内的波动率。基准波动率通常用于衡量资产相对于基准资产的风险水平，即测量**基准的风险性，波动越大代表基准风险越高**
算法：Benchmark Volatility = 标准差 / 基准价格 * 100% or sqrt([250/(n-1)]*sum([Rm-Rb]^2))
- 其中Rm为资产收益率，Rb为基准收益率，n为时间周期

2. 最大回测
定义：Maximum Drawdown，简称MDD，是指在选定周期内任一历史时点往后推，股价走到最低点时的收益率回撤幅度的最大值(即**在某一段时期内股价从最高点开始回落到最低点的幅度**)。**最大回撤用来描述买入股票后可能出现的最糟糕的情况**最大回撤是一个重要的风险指标
算法：MDD = (Px - P) / 最高点
- Px=策略某日股票和现金的总价
- Py=策略另一日股票和现金的总价

最大回撤回测：在聚宽回测界面单击左侧“最大回撤”可查看量化策略的最大回撤信息
