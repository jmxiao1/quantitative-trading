1，质量类因子介绍
质量类因子指与股票的财务质量，资本结构相关的因子。影响质量因子的因素大致包括:公司的盈利能力，盈利稳定性，资本结构，成长性，会计质量，派息/摊薄，投资能力等
- 指标：质量类因子包括净资产收益率，总资产收益率等

2，净资产收益率
净资产收益率是企业税后利润除以净资产得到的百分比率，该指标反映股东权益的收益水平，用以衡量企业运用自有资本的效率。该指标越高，说明企业的自有资本利用效率越好，当然，这并不是越高越好，因为过高的净资产收益率意味着企业可能存在资产减值的风险。
- 计算方法：净资产收益率=（净利润-优先股股利）/（期初净资产+期末净资产）/2 or 税后利润/所有者权益*100%
- 代码实现：get_fundamentals(query(indicator.roe).filter({查询条件}).order_by({升降序}),date='查询日期')

3，总资产收益率
总资产净利润反映的是公司运用全部资产所获得利润的水平，即公司每占用1元的资产能获得多少钱的利润，总资产净利润越高，表明公司投入产出水平越高，资产运营越有效，成本费用的控制水平越高。
- 计算方法：总资产收益率=（净利润+利息支出）/（期初总资产+期末总资产）/2 or 净利润/平均资产总额*100%
- 代码实现：get_fundamentals(query(indicator.roa).filter({查询条件}).order_by({升降序}),date='查询日期')
