1.价值类因子介绍
价值类因子即价值投资，是一种久经考验的投资策略。通过购买那种相对低价的股票，转化为在基本面标准度量股息，账面信息，利润，现金流或者其他公司价值的方法
指标：价值类因子包括市净率，市销率，(动态/静态)市盈率等

2.市净率
市净率可用于股票投资分析，一般来说市净率较低的股票，投资价值较高，相反，则投资价值较低。在判断投资价值时还要考虑当时的市场环境以及公司经营情况，盈利能力等因数
- 计算方法 市净率=总市值/净资产    or   每股市价/每股净资产
- 代码 get_fundamentals(query(valuation.pb_ratio).filter({查询条件})，date={查讯时间})

3.市销率
在国内证券市场，运用这一指标来选股可以剔除那些市盈率很低但主营业务没有核心竞争力而主要是依靠非经营性损益而增加利润的股票(上市公司)。该项指标既有助于考察公司收益基础的稳定性和可靠性，又能有效把握其收益的质量水平
- 计算方法 市销率=总市值/营业收入  or  股价/每股销售额
- 代码 get_fundamentals(query(valuation.ps_ratio).filter({查询条件})，date={查讯时间})

4.(动态/静态)市盈率
动态市盈率：指还没有真正实现的下一年度的预测利润的市盈率
静态市盈率：指已经实现的过去一年的实际利润的市盈率，表示该公司需要累积多少年的盈利才能达到如今的市价水平
- 计算方法动态市盈率=股票现价/未来每股收益预测值
- 计算方法 静态市盈率=股票现价/每股收益
市盈率越低，代表公司价值越高，投资回收期越短，风险越小，越值得投资
- 代码 get_fundamentals(query(valuation.pcf_ratio valuation.pe_ratio).filter({查询条件})，date={查讯时间})
