# 今日学习反趋向指标，包括RSI,WR,KDJ指标
-----
1. RSI指标
RSI即相对强弱指标，是期货市场和股票市场中最为著名的**摆动指标**。显示的是**股价向上波动的幅度占总的波动幅度的百分比**，如果其数值大，就表示市场处于强势状态，如果数值小，则表示市场处于弱势
- 计算方法：RSI=100-100/(1+RS)，其中RS=(N日内收盘价涨数之和)/(N日内收盘价跌数之和)
- 参数：一般取6，12，24
- RSI>80为超买，RSI<20为超卖
- RSI以50为中界线，RSI>50为强势，RSI<50为弱势
- RSI在50附近震荡，则市场处于平衡状态
- RSI在80以上形成M头或者头肩顶形态，则市场处于超买状态，股价将下跌，为向下反转信号
- RSI在20以下形成W底或者头肩底形态，则市场处于超卖状态，股价将上涨，为向上反转信号
- RSI向上t突破其高点连线时，买进；RSI向下突破其低点连线时，卖出

2. WR指标
WR指标即威廉指标(简称W%R)，它由拉瑞威廉首先发表。威廉指数是**摆动指标**，主要用于研究股价的波动，通过分析**股价波动**变化中的**峰与谷**决定买卖时机，是用来分享市场**短期**行情走势的技术指标
- 计算方法：WR=100*(HH-N)/(HH-L)，其中HH为N日内最高价，L为N日内最低价 or WR=(N日内最高价-当日收盘价)/(N日内最高价-N日内最低价)*100
- 参数：一般取6，10，14
- WR>80为超买，行情即将见底，买进
- WR<20为超卖，行情即将见顶，卖出

- 返回WR和MAWR  字典(dict):键(key)为股票代码，值(value)为WR值

3. KDJ指标
KDJ指标即随机指标，其在设计中充分考虑**价格波动的随机振幅**与**中短期波动**的测算，使其短期测市功能比移动平均线更加准确有效，在市场短期*超买超卖**的预测方面又比相对强弱指标敏感。
- 计算方法：RSV(未成熟随机指标值)=100*(C-LN)/(HN-LN)，其中C为当日收盘价，LN为N日内最低价，HN为N日内最高价
-          K=2/3*K1+1/3*RSV
-          D=2/3*D1+1/3*K
-          J=3*K-2*D
- 参数：一般取9，3，3
- 备注:若无前一日的K，D值，则K，D值取50，J值取100
- 指标>80时，回档几率大；指标<20时，反弹几率大
- K在20左右向上交叉D时
- K在80左右向下交叉D时
- J>100时，股价易反转下跌；J<0时，股价易反转上涨
- KDJ波动于50左右的任何信号，其作用都不大

- N:计算RSV的参数

- M2:计算D的M1日移动平均值
