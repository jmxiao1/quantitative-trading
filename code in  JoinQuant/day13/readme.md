1.净利润同比增长率
净利润指企业的税后利润 ，净利润增长率是衡量企业盈利能力的重要指标之一，净利润同比增长率是指企业净利润在连续两个会计年度之间的增长速度。净利润同比增长率越高，说明企业的盈利能力越强。
- 计算公式：(当期净利润 - 上期净利润) / 上期净利润的绝对值 * 100%     上期净利润指上一年度的同期净利润
- 代码实现：get_fundamentals(query(indicator.inc_net_profit_year_on_year).filter({查询条件}),date={查询时间})
2.净利润环比增长率
净利润环比增长率是指企业净利润在连续两个会计期间之间的增长速度。净利润环比增长率越高，说明企业的盈利能力越强。
注：环比增长率是针对上一期的，而同比增长率是针对上一年度同一期的。    
- 计算公式：(当期净利润 - 上期净利润) / 上期净利润的绝对值 * 100%     上期净利润指上一个月度的净利润
- 代码实现：get_fundamentals(query(indicator.inc_net_profit_annual).filter({查询条件}),date={查询时间})
3.营业利润率
营业利润率是指企业营业利润占营业收入的比例，是衡量企业盈利能力的重要指标之一。营业利润率越高，说明企业的盈利能力越强。
- 计算公式：营业利润 / 全部业务收入 * 100%、
- 代码实现：get_fundamentals(query(indicator.operation_profit_to_total_revenue).filter({查询条件}),date={查询时间})
4.销售净利润
销售净利润是指企业销售商品或提供劳务所获得的净利润，即企业实现净利润与销售收入的对比关系，是衡量企业在一定时期的盈利能力的重要指标之一。销售净利润越高，说明企业的盈利能力越强。
- 计算公式：净利润 / 销售收入 * 100%
- 代码实现：get_fundamentals(query(indicator.net_profit_margin).filter({查询条件}),date={查询时间})
5.销售毛利润
销售毛利润是指企业销售商品或提供劳务所获得的毛利润，即企业实现毛利润与销售收入的对比关系，是衡量企业在一定时期的盈利能力的重要指标之一。销售毛利润越高，说明企业的盈利能力越强。
- 计算公式：(销售净利润-产品成本) / 销售净收入 * 100%
